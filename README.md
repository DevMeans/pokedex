# Apuntes nest

1.  clonar el repositorio
2.  ejecutar

```
npm i
```

3. Tener nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```
5. Ejecutar seed para generar datos

```
localhost:3000/api/v2/seed
```
## Stack usado

- MongoDB
- Nest
